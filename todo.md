# Render Triangle Learning Project

This repo is a learning project for rendering the classic colored triangle through different graphics and platform APIs.

The assistant should help by:

- Explaining where each tool sits in the graphics stack.
- Guiding the user toward good documentation and learning resources.
- Reviewing code the user writes.
- Helping debug build, linking, runtime, and rendering issues.
- Creating files or implementations only when explicitly asked.

Persistent project direction lives in [ROADMAP.md](ROADMAP.md).

## Tool Folders

- [raylib](raylib/README.md)
- [OpenGL](opengl/README.md)
- [Vulkan](vulkan/README.md)
- [NVRHI](nvrhi/README.md)
- [WebGL](webgl/README.md)
- [WebGPU](webgpu/README.md)
- [SDL](sdl/README.md)
- [GLFW](glfw/README.md)
- [Sokol](sokol/README.md)

## Current First Step

Build and run the raylib hello triangle in `raylib/` after raylib is installed locally.
